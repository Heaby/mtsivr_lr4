# mtsivr_lr4
Программа для преобразования карт глубины в 3D модели с поддержкой моделей освещения

# Содержимое репозитория
depth_map_converter.py — основной код на Python

config.json — конфигурация программы

DepthMap_Test.dat — тестовые данные (фрагмент сферы)

DepthMap_12.dat — входные данные для варианта 12

output.vrml, output.obj, output.ply — результаты экспорта

output.bmp — BMP изображение с освещением

# Описание
Программа выполняет:

Чтение бинарных карт глубины (формат: double)

Преобразование в 3D полигональную сетку

Визуализацию с моделями отражения: Ламберта, Фонга-Блинна, Орена-Найара

Экспорт в форматы VRML/OBJ/PLY/BMP

## Зависимости
bash
pip install numpy PyOpenGL PyOpenGL_accelerate
## Управление в OpenGL
Стрелки — вращение камеры
W/S — приближение/отдаление
R — сброс вида к начальному положению
Q/Esc — выход из программы

## Формат входных данных
Карта глубины представляет собой бинарный файл со структурой:
height (double) — высота матрицы
width (double) — ширина матрицы
data (double[width*height]) — массив значений глубины

## Форматы вывода
VRML (.vrml) — основной формат 3D модели

OBJ (.obj) — совместим с Blender, Maya

PLY (.ply) — полигональный формат

BMP (.bmp) — растровое изображение с освещением

Совет: Если VRML не открывается в Blender, используйте imagetostl.com# mtsivr_lr4
